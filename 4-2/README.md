# im-cat

HTML/CSS/JS - 모달창/컴포넌트 기능 구현

## 👩‍💻 소개

저희집 고양이 딱지를 소개하는 사이트입니다.
사용자가 이름, 이미지, 내용을 입력시 컴포넌트가 추가됩니다.
이미지 미입력시 랜덤 이미지가 추가됩니다.

## 🔨 기술

- `HTML`
- `CSS`
- `JavaScript`

## 📌기능

1. 모달창 구현
2. 저장 버튼 클릭시 컴포넌트 추가
3. 이미지 미입력시 랜덤 이미지 추가
