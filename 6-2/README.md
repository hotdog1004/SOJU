## 🔨 기술

- `CRA`

- `JSX`

## 📌기능

1. 화면 크기에 따른 컴포넌트 위치 변경
2. 화면 크기에 따른 배경색 변경
3. 화면 크기에 따른 글씨 크기 변경
4. 컴포넌트가 넘칠 때 스크롤 처리.
