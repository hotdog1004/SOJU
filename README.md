# im-cat
HTML/CSS 기초 - 반응형 사이트 만들기


## 👩‍💻 소개
저희집 고양이 딱지를 소개하는 사이트입니다.

## 🔨 기술 
- `HTML`
- `CSS`

## 📌기능
1. 화면 크기에 따른 컴포넌트 위치 변경 
2. 화면 크기에 따른 배경색 변경
3. 화면 크기에 따른 글씨 크기 변경
4. 컴포넌트가 넘칠 때 스크롤 처리
